# OpportunityScannerScreen - Dual Model Enhancement

## Overview
The OpportunityScannerScreen now includes a **dual-model architecture** allowing you to switch between two signal-generation algorithms:

### Model Selection
At the top of the header card, you'll see a toggle between:
- **Original** - Classic model with proven fundamentals
- **Enhanced** - Advanced model with volatility clustering and multi-timeframe analysis (⚡)

## Model Comparison

### Original Model
**Logic:** EMA-based trend following with RSI and volume confirmation
- Uses 3-period EMAs (9, 21, 50)
- 60 candles minimum (300 minutes of 5m data)
- Basic volatility projection
- Horizon: 15-60 minutes based on confidence
- Expected Gain Range: 0.2% - 18%

**Best For:** 
- Stable market conditions
- Clear trend following
- Conservative traders

### Enhanced Model (⚡)
**Logic:** Multi-timeframe analysis with volatility clustering detection
- Uses 4-period EMAs (9, 21, 50, 200)
- 90 candles minimum (450 minutes of 5m data)
- Volatility ratio analysis comparing recent vs historical ATR
- Support/resistance detection
- Trend alignment bonus scoring
- Advanced momentum calculation (7-period + 14-period)
- Horizon: 15-60 minutes with volatility adjustment
- Expected Gain Range: 0.3% - 22%

**Best For:**
- Volatile market conditions
- Capturing volatility clustering
- Momentum breakouts
- Advanced traders

## Key Enhancements

### 1. Volatility Clustering Detection
The Enhanced model detects when recent volatility significantly differs from historical volatility, potentially signaling trading opportunities.

### 2. Multi-Timeframe Confirmation
- Short-term: EMA9 vs EMA21 (immediate trend)
- Medium-term: EMA21 vs EMA50 (structure)
- Long-term: EMA50 vs EMA200 (macro trend)
- Aligned trends get bonus points

### 3. Advanced RSI Bias
Enhanced RSI interpretation with extreme conditions:
- RSI < 20: Strong oversold signal
- RSI > 80: Strong overbought signal
- More nuanced zone detection

### 4. Momentum Layers
- 7-period momentum: Short-term price action
- 14-period momentum: Medium-term momentum
- Combined for better signal confirmation

## Model Badge
Each signal card displays the model used:
- **⚡** Yellow badge = Enhanced model (higher potential gain range)
- **○** Gray badge = Original model (conservative)

## Usage Tips

### When to Use Original
✓ Markets trending strongly in one direction
✓ Low volatility environments
✓ Risk-averse traders
✓ Stable pairs like major cryptos

### When to Use Enhanced
✓ Markets with sudden volatility spikes
✓ Before major events (when vol clusters)
✓ Breakout traders
✓ Pairs with variable volatility
✓ When you want to capture volatility premiums

## Scanning Details

The scan process automatically adjusts for the selected model:
- **Original Model:** Pulls 120 candles (1 hour of 5m data)
- **Enhanced Model:** Pulls 180 candles (1.5 hours of 5m data)

The enhanced model requires more historical data for accurate volatility clustering detection.

## Switching Models
Simply tap "Original" or "Enhanced" at the top of the screen before scanning. The next scan will use your selected model.

## Logic Preservation
**Important:** All original trading logic remains unchanged:
- Trade execution
- Entry/Exit/Stop calculations
- Risk management
- Trade evaluation and close conditions
- All other core functionality

Only the signal generation algorithm is different.
